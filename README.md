# Student Feedback Analyzer using Deep Learning

## Overview
A Deep Learning-based NLP application that analyzes student feedback and predicts sentiment as Positive, Neutral, or Negative.

## Features
- Text preprocessing
- Tokenization
- LSTM Deep Learning Model
- Sentiment Prediction
- Streamlit Web Application

## Technologies
- Python
- TensorFlow
- Keras
- NLP
- Streamlit
- Scikit-learn

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
